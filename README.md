# K_Nearest Neighbors From Scratch

This project demonstrates an implementation of **K_Nearest Neighbors From Scratch**
using Python, without relying on machine learning libraries like scikit-learn

## Dataset
- example as a Dictionary 

## Features

- KNN implementation
- Model training 

## Requirements
- Python 3.x
- NumPy
- matplotlib


## How to Run

k_nearest_neighbors_from_scratch.py
